## Descripción
Este proyecto es un simulador interactivo de estructuras cristalinas cúbicas de Bravais desarrollado en HTML, CSS y JavaScript.

Fue desarrollado como parte de la materia **Proyectos de Software Enfocados a Nanotecnología**.

El simulador permite visualizar estructuras cristalinas utilizadas en el estudio de materiales y nanotecnología.

## Estructuras incluidas
- Cúbica Simple (SC)
- Cúbica Centrada en el Cuerpo (BCC)
- Cúbica Centrada en las Caras (FCC)

## Funcionalidades
- Visualización interactiva de estructuras cristalinas
- Rotación del modelo 3D
- Ajuste de tamaño de átomos
- Exploración educativa de estructuras cristalinas

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
